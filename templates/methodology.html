Methodology

Step 1: Download Mininet VM. Install it in a virtualization software like VMware Worksstation, VMware Fusion, Oracle Virtualbox.

Step2: Download/clone the ryu-controller and application files from the Github repository onto mininet VM. Install the dependencies if required using 'sudo apt-get install package_name' or 'sudo pip install package_name'. Go to /ryu folder using 'cd ryu'. Install the controller using the following command:

$sudo python setup.py install

From the current ryu folder, to run the graphical user interface (that shows the topology) and the DFS application, run the following command:

$sudo ryu-manager ryu/app/gui_topology/gui-topology.py ryu/app/DFS/DFS.Py --observe-links --k-paths=1 --weight=delay

Step3: >Write a script in python for customised topology. The ip address of the remote controller (that it must connect to) is mininet VM's ip. Emulate the network in a separate terminal in mininet using the following command:

$sudo python 'customtopology_filename.py'

The results can be viewed in project poster
